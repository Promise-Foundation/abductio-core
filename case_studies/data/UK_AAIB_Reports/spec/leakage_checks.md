# Leakage Checks (No-Hindsight Gates)

## Disallowed language detector (soft gate)
- concluded
- cause
- probable
- resulted from
- therefore
- led to
- root cause
- the investigation found
- the report discusses
- safety action was taken because
- safety action
- has taken safety action
- report discusses
- caused
- led to

## Section allowlist gate (hard gate)
Evidence packet must be derived only from:
- factual header fields
- factual narrative
- witness statements explicitly tagged as such

Exclude by default:
- analysis
- conclusion
- safety actions
