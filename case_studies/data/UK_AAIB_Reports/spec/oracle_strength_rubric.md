# Oracle Strength Rubric (OS-1 .. OS-5)

## Scoring dimensions (0-2 each)
1) Evidence transparency
2) Independence / incentives
3) Methodological rigor
4) Mechanistic confirmation
5) Multi-oracle convergence

## Mapping score -> OS class
- OS-1: total 0-2
- OS-2: total 3-5
- OS-3: total 6-7
- OS-4: total 8-9 and convergence >= 1
- OS-5: mechanistic confirmation = 2 and total >= 8

## Required fields in oracle.md
- dimension scores + short rationale each
- final OS class + rationale
