# Oracle Strength Checklist (OS-1 .. OS-5)

## Assignment checklist
- [ ] Public record is thin/opaque -> OS-1
- [ ] Public record with documented facts -> OS-2
- [ ] Investigation with exhibits or structured analysis -> OS-3
- [ ] Multi-oracle convergence -> OS-4
- [ ] Mechanistic confirmation (tests/lab/verified failure) -> OS-5

## Required fields
- OS class
- 2-3 sentence rationale
- Confidence: high/medium/low
- Rater initials and date

## Mapping note
If a legacy single-letter strength is present:
- A -> OS-3
- B -> OS-2
- C -> OS-1
